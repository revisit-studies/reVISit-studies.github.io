ReactiveResponse

Defined in: parser/types.ts:474

The ReactiveResponse interface is used to define the properties of a reactive response. ReactiveResponses render as a list, that is connected to a WebsiteComponent, VegaComponent, or ReactComponent. When data is sent from the components, it is displayed in the list.

{
 "id": "reactiveResponse",
 "prompt": "Answer clicked in the stimulus",
 "location": "aboveStimulus",
 "type": "reactive"
}
